## 📖 Overview

As search engine landscapes evolve, traditional search (keywords and page rankings) is blending with **Answer Engine Optimization (AEO)** and **Generative Engine Optimization (GEO)**. Platforms like Perplexity, ChatGPT Search, Gemini, and Google AI Overviews retrieve, synthesize, and cite authoritative, fact-dense content. 

This playbook provides the specific prompts, role definitions, and guidelines needed to turn Anthropic's **Claude** into a Senior SEO Analyst, Schema Architect, and Content Engineer. It features:
*   **Structured Content Audits**: Quantify factual density and extractability.
*   **Topical Maps**: Establish semantic domain authority.
*   **JSON-LD Scripting**: Auto-generate Schema markup for rich snippets.
*   **Search Console Analysis**: Analyze GSC exports for cannibalization and CTR wins.

## 🛠️ How to Use

The prompts in this playbook are designed to be copied and pasted directly into Claude. For the best workflow, we recommend using them inside **Claude Projects** or as **Custom Instructions**:

### Option 1: Claude Projects (Recommended for teams/ongoing work)
1. Log into your Claude Pro/Team account and create a new **Project**.
2. Copy the system prompt from **Chapter 9: The Claude Project Workspace Setup** and paste it into the project's **Set Custom Instructions** panel.
3. Export your company's core assets (e.g. landing page copy, value propositions, about pages) and upload them to the project's **Files** area.
4. Copy individual chapter prompts (like the *Technical Audit Prompt* or *Schema Builder*) whenever you need to execute specific tasks.

### Option 2: Claude Custom Instructions
1. Open Claude and go to **Settings** -> **Custom Instructions**.
2. Paste the **Chapter 2: Brand Context Mapping** setup or **Chapter 9: Custom Instructions** into the settings to make Claude act as your permanent SEO analyst across all standard chat threads.

### Option 3: Quick copy-paste
1. Click the **Copy Prompt** button on any prompt card in the [Live Playbook Webpage].
2. Replace the highlighted parameters in brackets (e.g. `[Paste Page Content Here]`) with your own details.
3. Send it to Claude and get immediate, customized deliverables.

### 🌟 Option 4: All-in-One Master Prompt (All 10 Skills Combined)
If you want Claude to have access to all 10 skills simultaneously in a single workspace or session:
1. Open or download the **[Master All-in-One Prompt](all-in-one/master_seo_aeo_prompt.md)**.
2. Paste the prompt text into your **Custom Instructions** or upload the file directly into your **Claude Project Knowledge Base**.
3. Trigger any specific skill by using slash commands (e.g., `/aeo`, `/brand`, `/topical`, `/schema`, `/gsc`) directly in your chat!
